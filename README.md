# application-support-incident-SLA-analytics

## Overview
An Excel-based dashboard created to analyze simulated application support incidents and SLA performance.

## Objective
To help support teams monitor incidents, identify critical issues, track SLA compliance, analyze resolution time, and find recurring root causes.

## Tools Used
- Microsoft Excel
- PivotTables
- PivotCharts
- Slicers
- Excel formulas

## Key KPIs
- Total Incidents
- SLA Compliance Percentage
- Average Resolution Time
- Critical P1 Incidents
- Total Users Affected

## Analysis Performed
- Incidents by application
- Incident distribution by priority
- SLA breaches by application
- Average resolution time by category
- Top root causes
- Incident trend over time

## Key Learnings
- Incident management
- Ticket prioritization
- SLA tracking
- Root cause analysis
- Escalation awareness
- Support analytics
